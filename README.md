# Smart Todo App

A modern and interactive **Smart Todo Application** built using **HTML, CSS, and JavaScript**.
This project helps users manage their daily tasks efficiently with features like task filtering, editing, dark mode, and local storage.

The application has a clean and responsive user interface designed to provide a smooth productivity experience on both desktop and mobile devices.


# Project Description

The Smart Todo App is an advanced version of a basic todo list.
Users can easily **add, edit, delete, and manage tasks**, as well as **mark them as completed or pending**.

The app also includes **task filtering options and dark mode support**, making it both functional and visually appealing. All tasks are stored in the browser using **localStorage**, so the data remains saved even after refreshing the page.


##  Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑 Delete tasks
* ✅ Mark tasks as completed
* 🔍 Filter tasks (All / Completed / Pending)
* 💾 Data saved using **Local Storage**
* 🌙 Dark Mode / Light Mode toggle
* 📱 Responsive design for desktop, tablet, and mobile
* 🎨 Modern and clean user interface
* ⚡ Smooth animations for better user experience
